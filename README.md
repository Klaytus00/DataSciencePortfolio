# Data Analysis and ML Projects

A collection of personal projects where I explored real datasets and built predictive models. Each one started with a question I actually wanted to answer.

---

## Projects

### 1. What Drives Sale Prices: Size or Location?
**File:** `Housing Price Analysis.ipynb`

Looked at U.S. housing data to figure out whether property size or location has more influence on sale prices. Also checked which states have the priciest real estate markets and whether home size and price move together.

**Tools:** Python, Pandas, Matplotlib, Seaborn

---

### 2. House Price Prediction
**File:** `House Price Prediction.ipynb`

Built a model to predict house prices by city using Ridge Regression. Focused on cleaning the data and tuning the model to reduce overfitting.

**Tools:** Python, Scikit-learn, Ridge Regression

---

### 3. Used Cars Price Prediction
**File:** `Used Cars Price Prediction.ipynb`

Predicted used car prices based on attributes like mileage, brand, and year using Ridge Regression. Started as a Kaggle competition entry.

**Tools:** Python, Scikit-learn, Ridge Regression

---

### 4. FIFA World Cup Analysis (1930–2022)
**File:** `Fifa World Cup Analysis 1930 - 2022.ipynb`

Analyzed match outcomes, goal patterns, manager performance, attendance trends, and team rankings across 90+ years of World Cup data. Also built a model to predict match results.

**Tools:** Python, Pandas, Seaborn, Scikit-learn (Random Forest, Decision Tree)

---

### 5. YouTube Content Performance Analysis
**File:** `Youtube Content Performance Analysis.ipynb`

Dug into which video content types perform best by watch time, which editors drive the most views, and which editor-content combinations get the highest CTR.

**Tools:** Python, Pandas, Matplotlib, Seaborn

---

## Stack

All projects run in Jupyter Notebook and use the standard Python data stack — Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.
